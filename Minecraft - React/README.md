This program replicates Minecraft using **Javascript, React and Three.js**.

**Program Information:**
- There are 5 types of blocks you can use: Grass, Wood, Log, Glass and Dirt.
- Switch the blocks by using the digits 1-5 on your keyboard.
- Move around using your mouse and the WASD keys, along with the space bar to jump.
- Click to add blocks, and use Option/Alt+Click to remove blocks.
- You can save the world in your local storage by using the Menu options at the top left of the screen.

**To Run the Game:**
1. Go to the Terminal/Command Prompt.
2. Make sure you are in the correct directory: Ex. "cd Desktop/Projects/Minecraft"
3. Call "npm install && npm start"
4. Have fun!
